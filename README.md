## Installation

Using [lazy.nvim](https://github.com/folke/lazy.nvim):
```lua
{
  "sahilNaran/bo-banana-bread.nvim",
  priority = 1000,
  config = function()
    require("bo-banana-bread").setup()
  end,
}
```

Using [packer.nvim](https://github.com/wbthomason/packer.nvim):
```lua
use({
  "sahilNaran/bo-banana-bread.nvim",
  config = function()
    require("bo-banana-bread").setup()
  end
})
```
